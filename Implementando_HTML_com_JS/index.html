Validação - JavaScript

Você terá a simples missão de digitar um número em cada campo do formulário abaixo. Porém, se atente ao seguinte critério: o formulário só será válido se o número digitado no campo A for menor que o número digitado no campo B!

Após terminar de preencher, clique no botão validar para verificar se seus números são válidos ou não!
Por favor, teste todas as possibilidades, nunca se sabe o que você poderá encontrar!!!

Erro! O número do campo A não pode ser maior que o número do campo B.

Atenção! O número do campo A não pode ser igual ao número do campo B.
